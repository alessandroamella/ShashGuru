INSTRUZIONI PER INSTALLAZIONE SOFTWARE SHASHGURU

1. Installare docker desktop ed avviarlo 
    IMPORTANTE: Nelle impostazioni di docker, una volta avviato, spuntare "Start Docker Desktop when you sign in to your computer"
2. Installare git

3. Copiare cartella ShashGuru da USB a Desktop

4. docker compose -f docker-compose-ai-pc.yml build

Avvio

1. Icona di windows -> Barra di Ricerca -> Command Prompt
2. Eseguire (cliccare su) START.bat
3. Andare all'indirizzo http://localhost

Avvio di backup nel caso ci siano problemi:

1. Aprire un browser (Edge/Chrome/Firefox)
2. Andare all'indirizzo https://shashguru.disi.unibo.it/
